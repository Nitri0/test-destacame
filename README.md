# Destacame Test

## Requerimientos
* Docker
* Docker Compose



## Build

1. Clonar repositorio
```
    $ git clone https://github.com/Nitri0/test-destacame.git
```

2. Entrar carpeta de proyecto
```
    $ cd test-destacame
```

. Levantar contendores
```
    $ docker-compouse up
```

## Acceso

Nginx
```
    localhost:5888
```

Angular dev
```
    localhost:4200
```
